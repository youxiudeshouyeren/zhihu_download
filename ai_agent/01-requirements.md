**文档版本**: v1.0
**创建时间**: 2026-03-13
**最后更新**: 2026-03-13

---

## 1. 项目概述

### 1.1 项目目标

开发一款**纯本地运行、无服务端依赖**的知乎个人收藏夹导出工具，支持：
- 命令行（CLI）交互
- Web UI 交互（后续扩展）
- MCP Skill 接口（后续扩展）

### 1.2 核心功能

1. **Cookie 授权** - 加密存储，安全验证
2. **收藏夹管理** - 列表展示、选择导出
3. **内容抓取** - 回答、专栏文章抓取
4. **格式转换** - Markdown、PDF 导出
5. **进度管理** - 实时进度、失败重试

### 1.3 合规红线

- ⚠️ **禁止**抓取盐选付费内容全文
- ⚠️ **禁止**高频并发请求
- ⚠️ **禁止**数据上传第三方服务器
- ⚠️ **禁止**明文存储 Cookie

## 2. 功能需求

### 2.1 授权模块

| 功能 | 描述 |
|------|------|
| Cookie 输入 | 支持字符串输入、JSON 文件加载 |
| Cookie 验证 | 调用知乎 API 验证有效性 |
| 加密存储 | 使用 PBKDF2+FERNET 加密本地存储 |
| 一键清除 | 支持清除保存的授权信息 |

### 2.2 收藏夹模块

| 功能 | 描述 |
|------|------|
| 列表获取 | 拉取用户所有收藏夹 |
| 信息展示 | 名称、创建者、内容数 |
| 选择导出 | 支持指定收藏夹 ID 导出 |

### 2.3 内容抓取模块

| 功能 | 描述 |
|------|------|
| 回答抓取 | 解析 HTML，保留排版 |
| 专栏抓取 | 解析 HTML，保留排版 |
| 图片下载 | 本地化保存 |
| 请求延迟 | 1-3 秒随机延迟 |

### 2.4 格式转换模块

| 格式 | 描述 |
|------|------|
| Markdown | Obsidian 风格，图片本地引用 |
| PDF | weasyprint 渲染，A4 排版，页眉页脚 |

### 2.5 导出控制模块

| 功能 | 描述 |
|------|------|
| 进度展示 | Rich 进度条 |
| 失败处理 | 记录日志，继续执行 |
| 版权声明 | 每篇内容添加版权声明 |

## 3. 数据结构

### 3.1 收藏夹结构

```json
{
  "id": "收藏夹 ID",
  "title": "收藏夹名称",
  "creator": {"name": "创建者"},
  "count": 内容数量
}
```

### 3.2 内容结构

```json
{
  "url": "内容链接",
  "title": "标题",
  "html": "HTML 内容",
  "type": "answer|article"
}
```
