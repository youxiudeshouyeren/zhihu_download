# Web UI 专业深色模式重构总结

**日期**: 2026-03-14
**版本**: v2.0

---

## 重构目标

按照专业 UI/UX 规范重新设计知乎收藏夹导出工具的 Web 界面，解决以下核心问题：
- 信息架构逻辑混乱
- 视觉层级缺失
- 控件规范不统一
- 深色模式适配错误

---

## 完成事项

### 1. 专业深色模式色彩体系

| 变量 | 旧值 | 新值 | 说明 |
|------|------|------|------|
| `--bg-body` | `#0a0e14` | `#0F172A` | 深空蓝灰背景，非纯黑避免视觉疲劳 |
| `--bg-card` | `#151a23` | `#1E293B` | 岩灰色卡片背景 |
| `--primary-500` | `#1890ff` | `#165DFF` | 专业蓝色，沉稳不刺眼 |
| `--success-color` | `#52c41a` | `#00B42A` | 鲜绿色成功态 |
| `--border-color` | `#2d3748` | `#334155` | 板岩灰边框 |
| `--text-primary` | `#ffffff` | `#FFFFFF` | 纯白标题色 |
| `--text-secondary` | `#a0aec0` | `#E2E8F0` | 浅灰正文色 |
| `--text-muted` | `#718096` | `#94A3B8` | 中灰辅助色 |

### 2. 间距规范（8px 原则）

新增 CSS 变量：
```css
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 12px;
--spacing-lg: 16px;
--spacing-xl: 24px;
--spacing-2xl: 32px;
```

应用规则：
- 卡片内边距：统一 `16px`
- 模块/卡片之间间距：统一 `24px`
- 元素之间间距：`8px/12px`

### 3. 圆角规范

```css
--radius-sm: 4px;
--radius-md: 8px;
--radius-lg: 12px;
--radius-xl: 16px;
```

### 4. 布局结构优化

- **侧边栏**: 固定宽度 `280px`，通栏到底
- **主内容区**: 自适应宽度，左侧留白 `280px`
- **页面切换**: 淡入动画 `fadeIn 0.3s`
- **导出栏**: 底部固定，通栏显示

### 5. 控件样式统一

#### 按钮分级

| 类型 | 背景 | 文字 | 边框 | 阴影 |
|------|------|------|------|------|
| 主按钮 | `linear-gradient(135deg, #165DFF, #0050cc)` | 白色 | 无 | `0 4px 14px rgba(22, 93, 255, 0.4)` |
| 次按钮 | `#334155` | `#E2E8F0` | `1px solid #334155` | 无 |
| 危险按钮 | `linear-gradient(135deg, #ff3333, #cc0000)` | 白色 | 无 | `0 4px 14px rgba(255, 51, 51, 0.4)` |

#### 输入框规范

- 常态：`border: 1px solid #334155`
- 聚焦态：`border-color: #165DFF` + `box-shadow: 0 0 0 3px rgba(22, 93, 255, 0.15)`

#### 复选框规范

- 使用浏览器原生 `accent-color: #165DFF`

### 6. 组件样式更新

#### 侧边栏导航
- 激活态：左侧 `3px` 强调线 + 蓝色背景光晕
- hover 态：背景变亮 + 轻微位移

#### 状态指示器
- 脉冲动画：`animation: pulse 2s infinite`
- 有效状态：绿色光晕
- 无效状态：红色光晕

#### 卡片组件
- 圆角：`16px`
- 悬停效果：边框高亮 + 阴影加深

#### 任务卡片
- 进度条流光动画：`shimmer 2s infinite`
- 状态标签：彩色背景 + 大写文字

#### 通知弹窗
- 右上角滑入：`slideInRight 0.3s`
- 分类颜色：成功绿 / 错误红 / 信息蓝
- 背景模糊：`backdrop-filter: blur(8px)`

### 7. 交互细节优化

#### 加载状态
- 模块内显示 spinner + 文字提示
- spinner 旋转动画：`spin 1s linear infinite`

#### 空状态
- 图标 + 主文案 + 辅助文案（可选）
- 居中对齐，灰色文字

#### hover 态反馈
- 收藏夹项：边框高亮 + 向右位移 `4px`
- 文章项：边框高亮 + 向右位移 `4px`
- 图标按钮：背景变亮 + 缩放 `1.05`

#### 选中态反馈
- 收藏夹项：蓝色光晕 + 背景色
- 导航项：左侧强调线 + 背景光晕

### 8. 响应式设计

#### 桌面端（> 1200px）
- 侧边栏固定显示
- 导出栏左侧留白 `280px`

#### 平板端（768px - 1200px）
- 侧边栏隐藏，需手动展开
- 导出栏通栏显示

#### 移动端（< 768px）
- 主内容区留白缩小
- 导出栏垂直布局
- 工具栏垂直布局
- 输入框全宽显示
- 按钮全宽显示

### 9. 视觉效果

| 效果 | 代码 | 应用场景 |
|------|------|----------|
| 光晕 | `0 0 24px rgba(22, 93, 255, 0.4)` | 主按钮、导航激活态 |
| 渐变 | `linear-gradient(135deg, #165DFF, #0050cc)` | 主按钮、进度条 |
| 流光 | `shimmer 2s infinite` | 进度条 |
| 脉冲 | `pulse 2s infinite` | 状态指示器 |
| 淡入 | `fadeIn 0.3s` | 页面切换 |
| 滑入 | `slideInRight 0.3s` | 通知弹窗 |
| 上升 | `slideUp 0.3s` | Modal |

## 技术实现

### CSS 变量体系

```css
:root {
    /* 颜色 */
    --bg-body: #0F172A;
    --bg-card: #1E293B;
    --primary-500: #165DFF;
    --success-color: #00B42A;

    /* 间距 */
    --spacing-lg: 16px;

    /* 圆角 */
    --radius-lg: 12px;

    /* 阴影 */
    --shadow-xl: 0 12px 48px rgba(0, 0, 0, 0.6);
    --glow-primary: 0 0 24px rgba(22, 93, 255, 0.4);

    /* 渐变 */
    --gradient-primary: linear-gradient(135deg, #165DFF 0%, #0050cc 100%);
}
```

### 动画曲线

- 标准过渡：`0.25s cubic-bezier(0.4, 0, 0.2, 1)`
- 缓动过渡：`0.3s ease`
- 线性过渡：`1s linear`
