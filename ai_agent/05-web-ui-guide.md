## 功能说明

### 1. 收藏夹管理页面

**功能**: 添加和管理要导出的收藏夹

**操作步骤**:

1. **输入收藏夹 URL 或 ID**
   - 支持完整 URL: `https://www.zhihu.com/collection/998150123`
   - 支持短 URL: `www.zhihu.com/collection/998150123`
   - 支持纯 ID: `998150123`

2. **点击"添加"按钮**
   - 系统自动解析收藏夹信息
   - 显示收藏夹名称、创建者、文章数量

3. **选择收藏夹**
   - 点击收藏夹卡片进入文章列表页面
   - 右侧箭头按钮可删除收藏夹

### 2. 文章列表页面

**功能**: 查看收藏夹文章列表，选择要导出的文章

**操作步骤**:

1. **选择收藏夹**
   - 从顶部收藏夹名称选择器中选择

2. **选择文章**

     - **取消全选**: 取消所有勾选
     - **反选**: 反转当前选择状态

3. **查看文章信息**
   - 文章标题
   - 文章类型（回答/文章）
   - 点赞数、评论数
   - 作者信息

### 4. 设置页面

**功能**: 配置 Cookie 和导出选项

#### Cookie 配置

**Cookie JSON 格式**:
```json
[
  {"name": "z_c0", "value": "abc123..."},
  {"name": "__zhd", "value": "xyz789..."},
  {"name": "_xsrf", "value": "def456..."}
]
```

**获取 Cookie 文件**:

1. 使用浏览器开发者工具导出 Cookie
2. 或使用现有项目的 `cookies.json` 文件
3. 直接粘贴到文本框

**操作按钮**:
- **保存 Cookie**: 保存并验证 Cookie
- **加载当前 Cookie**: 从服务器加载已保存的 Cookie
- **清除 Cookie**: 删除已保存的 Cookie

**状态指示器**:
- 🟢 已连接：Cookie 有效
- 🟡 未验证：Cookie 未验证
- 🔴 未连接：Cookie 无效或缺失

#### 导出目录配置

**输入框**: 输入输出目录路径

**支持的格式**:
- 相对路径：`./downloads`
- 绝对路径（Linux/macOS）：`/home/user/zhihu_exports`
- 绝对路径（Windows）：`D:\zhihu_exports`

## 常见问题

### Q1: 页面无法加载？

**A**: 确保 Web UI 服务器正在运行：

```bash
python web/server.py
```

检查控制台是否有报错信息。

### Q2: Cookie 保存后仍显示"未连接"？

**A**: Cookie 可能已过期或格式不正确。

1. 重新获取 Cookie
2. 确保 JSON 格式正确
3. 检查浏览器控制台是否有错误

### Q3: 文章列表加载失败？

**A**: 可能是以下原因：

1. Cookie 无效 - 重新保存 Cookie
2. 收藏夹 ID 错误 - 检查 ID 是否正确
3. 网络连接问题 - 检查网络状态

### Q4: 导出任务卡住不动？

**A**: 可能是网络请求超时或内容过多。

1. 查看浏览器控制台是否有错误
2. 刷新页面查看任务状态
3. 检查日志文件 (`downloads/collection_<ID>/logs/`)

### Q5: 如何停止正在进行的导出任务？

**A**: 关闭浏览器标签页即可停止导出。

已导出的内容不会丢失，下次可使用 `--resume` 参数继续。

## 技术细节

### 前端技术

- **框架**: 原生 JavaScript (ES6+)
- **样式**: 自定义 CSS (深色模式)
- **状态管理**: 全局变量 + localStorage

### 后端技术

- **框架**: FastAPI
- **模板**: Jinja2
- **API**: RESTful 风格

### API 接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/cookies` | POST | 保存 Cookie |
| `/api/cookies` | GET | 获取 Cookie 状态 |
| `/api/cookies` | DELETE | 清除 Cookie |
| `/api/collections` | GET | 获取收藏夹列表 |
| `/api/collections/{id}/articles` | GET | 获取文章列表 |
| `/api/export/{id}` | POST | 创建导出任务 |

## 最佳实践

1. **首次使用**: 先在设置页面配置 Cookie
2. **批量导出**: 使用 CLI 命令行更适合大批量导出
3. **交互式选择**: Web UI 适合选择特定文章导出
4. **长时间导出**: 建议使用 CLI，可在后台运行

---

## 与 CLI 对比

| 功能 | Web UI | CLI |
|------|--------|-----|
| 图形界面 | ✅ | ❌ |
| 文章选择 | ✅ 可视化 | ❌ 全部导出 |
| 批量处理 | ⚠️ 需保持浏览器打开 | ✅ 后台运行 |
| 断点续传 | ⚠️ 刷新页面后丢失 | ✅ 自动检测 |
| 脚本集成 | ❌ | ✅ |

**推荐场景**:
- **Web UI**: 交互式选择少量文章
- **CLI**: 批量导出、自动化脚本、服务器环境
